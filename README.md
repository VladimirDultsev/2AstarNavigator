# 2AStar Pathfinding System
Двунаправленный алгоритм A* для поиска оптимальных маршрутов на графе дорог с интерфейсом Node.js.

## Описание проекта
Система для поиска кратчайшего пути между двумя географическими точками с использованием двунаправленного алгоритма A* (2A*). 
Состоит из двух компонентов:
1. C++ модуль - высокопроизводительная реализация алгоритма
2. Node.js интерфейс - удобный API для взаимодействия с C++ модулем

## Особенности
Двунаправленный поиск A* - одновременный поиск от старта к финишу и от финиша к старту

Точное вычисление расстояния по гаверсинусной формуле

Формат GeoJSON - работа с географическими данными в самом популярном формате

Асинхронный API - неблокирующее взаимодействие через Node.js

Кэширование графа - граф загружается единожды при запуске

## Структура проекта
main.cpp - реализация алгоритма поиска на C++
index.js - Node.js API для удобной интеграции кода в Ваши проекты

# Формат данных
## Формат графа
Формат GeoJSON с дорожной сетью
```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "geometry": {
        "type": "LineString",
        "coordinates": [[37.615555, 55.752220], [37.615600, 55.752300]] // Рёбра графа
      }
    }
  ]
}
```
## Формат выходных данных
Формат GeoJSON с маршрутом
```json
{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {},
      "geometry": {
        "type": "LineString",
        "coordinates": [[37.615555, 55.752220], ...] // Рёбра найденного пути 
      }
    }
  ]
}
```
